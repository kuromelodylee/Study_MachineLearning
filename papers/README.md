#딥러닝 분석
1. 과제: 딥러닝 네트워크 조사 보고서
2. 모델: 다층퍼셉트론(Multilayer Perceptron(MLP))과 합성곱신경망(Convolutional Neural Network(CNN)
3. 내용: MLP와 CNN의 등장배경, 적용분야, 네트워크의 구성 요소, 학습 파라미터와 하이퍼 파라미터, 학습에 사용되는 알고리즘 등 전반적인 내용에 대한 조사 결과
